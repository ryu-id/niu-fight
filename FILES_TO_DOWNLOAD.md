# 📦 FILE YANG HARUS DI-DOWNLOAD

## File Utama untuk Hermes Agent

**Download seluruh folder `niu-fight` ini** (bisa dalam bentuk ZIP atau clone repo).

### File Penting yang Harus Ada:

1. `project.godot` ← Konfigurasi utama
2. `export_presets.cfg` ← Preset Android
3. `HERMES_EXECUTION_PROMPT.md` ← Prompt lengkap untuk Hermes
4. Semua folder `scenes/`, `scripts/`, `assets/`

---

## Cara Download

### Opsi 1: Download ZIP dari GitHub
https://github.com/ryu-id/niu-fight/archive/refs/heads/master.zip

### Opsi 2: Git Clone
```bash
git clone https://github.com/ryu-id/niu-fight.git
```

---

## Setelah Download

1. Extract / buka folder `niu-fight`
2. Copy seluruh isi folder ke Hermes Agent
3. Berikan file `HERMES_EXECUTION_PROMPT.md` kepada Hermes Agent
4. Hermes Agent akan menjalankan semua langkah sesuai prompt

---

**Tidak perlu download file lain.** Semua yang dibutuhkan sudah ada di folder ini.
