# Niu-Fight — Arena.ai

> Fighting game Android bertema **Nusantara** (Kerajaan Nusantara).
> Dibangun dengan **Godot 4.3**, target **Android (API 24+)**.

## Karakter

| Karakter | Gaya Bertarung |
|---|---|
| **Gajah Mada** | Grappler — cengkeraman mematikan |
| **Naga Samudra** | Zoner — serangan jarak jauh air |
| **Garuda Singhasari** | Aerial — serangan udara cepat |
| **Keris Emas** | Precision — serangan tajam presisi |

## Cara Build APK

```bash
# Prasyarat: Godot 4.3 + Android SDK + JDK 17
# Export dari Godot Editor:
#   Project → Export → Android → Export & Run
```
