## 🚀 CARA BUILD APK (Langsung Bisa)

### Langkah 1: Buka di Godot 4.3
1. Download Godot 4.3 (godotengine.org)
2. Buka folder `niu_fight`

### Langkah 2: Export Android
1. Project → Export
2. Pilih preset **Android**
3. Klik **Export Project**
4. Simpan sebagai `Niu-Fight.apk`

### Langkah 3: Install di HP
- Copy APK ke Infinix GT 30 Pro
- Install (aktifkan Unknown Sources)
- Main game
